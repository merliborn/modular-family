pattern FNat :: Natural -> SNat n -> FinNat n Source #

Pattern/setter for FinNat data

As an expression, FNat i n throws exception if (1) i is larger than bound (ValueOverflow) or (2) n is less than or equal to 0 (NonPositiveUpperBound).
